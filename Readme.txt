# 💼 Personal Portfolio Website 🌐

Welcome to my **Personal Portfolio Website**!  
This project showcases my skills, projects, and contact information, built with modern web technologies to give a clean, responsive, and interactive experience. ✨

---

## 🚀 Technologies Used

- 🧱 **HTML5** – Markup structure
- 🎨 **CSS3** – Styling and layout
- ⚙️ **JavaScript** – Interactive elements
- 🅱️ **Bootstrap 4/5** – Responsive design and components

---

## 🔍 Features

✅ **Fully Responsive Design** (Mobile-Friendly)  
✅ **Smooth Scroll Navigation**  
✅ **About Me Section**  
✅ **Skills / Technologies**  
✅ **Project Showcase / Portfolio Cards**  
✅ **Contact Form** with basic validation  
✅ **Social Media Links** with icons  
✅ **Downloadable Resume** (optional)
